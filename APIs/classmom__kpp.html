Detailed Description

This module provides the K-profile parameterization (aka KPP) of Large et al., 1994 (ftp://128.171.154.9/pub/rlukas/Thibault/Large%20etal%20RevGeo%201994.pdf). KPP is implemented via the CVMix package (https://code.google.com/p/cvmix) which this module invokes.
